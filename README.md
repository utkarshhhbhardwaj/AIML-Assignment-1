# AIML Assignment 1 – Student Exam Score Prediction

## Problem Statement
Predict student exam scores using study hours and sleep hours.

## Dataset
The dataset contains:
 Hours_Studied,
 Sleep_Hours,
 Score

## Model Used
Linear Regression (Scikit-learn)

## Results
Model performance was evaluated using:
- MAE
- R2 Score

## Conclusion
Study hours significantly affect exam performance. The model predicts scores reasonably well.
